# Alkoholverktyg (3 sidor)

Filer:
- index.html – meny / start
- alkohol-per-burk.html – räknar ren alkohol per burk (cl och gram)
- promille.html – grov promille-estimator (Widmark-ish)
- pris-alkohol-per-krona.html – pris per burk inkl. omkostnader + alkohol per krona (sparar lokalt)

## GitHub Pages
1. Lägg alla filer i repo-root
2. Settings → Pages → Deploy from branch → main / (root)

## Notis
Promille-sidan är en uppskattning och ska inte användas för beslut om bilkörning.
